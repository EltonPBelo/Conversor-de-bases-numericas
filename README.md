# Tema 4: Conversor de bases numéricas
Atividade da matéria de versionamento de codigo

Um aplicativo web simples, rápido e responsivo para conversão de bases matemáticas em tempo real. Não necessita de backend para funcionar (Single-Page Application).

## 🚀 Funcionalidades

- **Conversão em Tempo Real**: Os valores são convertidos instantaneamente enquanto você digita.
- **Múltiplas Bases Suportadas**:
  - Decimal (Base 10)
  - Binário (Base 2)
  - Octal (Base 8)
  - Hexadecimal (Base 16)
- **Validação de Entrada**: O sistema alerta visualmente caso um caractere inválido para aquela base seja digitado.
- **Suporte a Números Grandes**: Utiliza `BigInt` no JavaScript para evitar os limites de precisão com números muito extensos.

# Tema 4: Conversor de bases numéricas
Atividade da matéria de versionamento de codigo

Um aplicativo web simples, rápido e responsivo para conversão de bases matemáticas em tempo real. Não necessita de backend para funcionar (Single-Page Application).

## 🚀 Funcionalidades

- **Conversão em Tempo Real**: Os valores são convertidos instantaneamente enquanto você digita.
- **Múltiplas Bases Suportadas**:
  - Decimal (Base 10)
  - Binário (Base 2)
  - Octal (Base 8)
  - Hexadecimal (Base 16)
- **Validação de Entrada**: O sistema alerta visualmente caso um caractere inválido para aquela base seja digitado.
- **Suporte a Números Grandes**: Utiliza `BigInt` no JavaScript para evitar os limites de precisão com números muito extensos.


## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura da página.
- **CSS3**: Estilização moderna, responsiva e feedback visual de erros.
- **JavaScript (Vanilla)**: Toda a lógica matemática e manipulação da interface (DOM) sem uso de bibliotecas ou frameworks externos.

## 📂 Estrutura do Projeto

O projeto foi dividido em três arquivos principais para melhor organização e boas práticas de desenvolvimento:

- `index.html`: Contém a estrutura e a interface visual.
- `style.css`: Contém todo o design e estilização do cartão de conversão.
- `script.js`: Contém o motor de cálculo e a lógica de validação.
